从主机和应用获取日志

EventLog Analyzer可以获取以下类型的日志：

主机:

  • Windows主机
  • Linux/ Unix主机
  • 思科设备
  • Syslog设备
  • IBM AS/400 (iSeries)

应用:

  • MS IIS W3C Web服务器
  • MS IIS W3C FTP服务器
  • Apache Web服务器
  • DHCP Windows服务器
  • DHCP Linux服务器
  • 打印服务器
  • MS SQL数据库服务器
  • Oracle数据库服务器
说明：通过“通用日志处理和索引”技术，EventLog Analyzer也支持其他没有列出的机器或应用，只要这些接收或导入的日志是人可读的（没有加密）格式。

仪表板的欢迎屏幕指导你添加主机和应用去获取日志。欢迎屏幕的选项有：

  • 如何添加Windows主机收集事件日志，添加Linux、Unix-host、其他Syslog设备和思科设备收集syslogs
  • 如何导入应用日志
