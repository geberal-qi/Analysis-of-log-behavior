如何安装和卸载EventLog Analyzer

  • 如何安装EventLog Analyzer?
  • 如何卸载EventLog Analyzer?

如何安装？

如果要在Windows系统中安装32位的EventLog Analyzer，运行ManageEngine_EventLogAnalyzer.exe文件；如果在32位的Linux系统中安装，运行ManageEngine_EventLogAnalyzer.bin文件。

如果要在Windows系统中安装64位的EventLog Analyzer，运行ManageEngine_EventLogAnalyzer_64bit.exe文件；如果在64位的Linux系统中安装，运行ManageEngine_EventLogAnalyzer_64bit.bin文件。

启动安装程序后，有如下两个选项：

  • 一键安装
  • 高级安装

选择一键安装选项来快速安装，这意味着你同意产品的许可协议。产品将安装到C:\ManageEngine\EventLog文件夹。使用8400作为Web服务器的端口。并安装为服务。

选择高级安装选择来定制产品安装。安装向导将指导你逐步完成安装。

高级安装步骤

  • 同意产品协议的条款。你可以打印该协议，或者保存到其他地方以供参考。
  • 选择要安装的产品版本：独立安装版本、分布式安装版本或免费版本。

    独立安装版本适用于中小型企业 - 如果你是一个中小型企业，并只在一个地方管理日志，管理的日志设备或应用不超过600个，独立安装版本就适合你。

    分布式版本适用于大型的企业 - 如果你是一个大型的企业，在多个地方管理日志，管理的日志设备或应用超过12000个，那么请使用分布式版本。

    免费版本 - 如果你的规模非常小，管理不超过5个日志设备或应用，你可以下载Evenglog Analyzer的独立版本安装为免费版。

  • 通过浏览按钮选择安装产品的文件夹。默认的安装位置为C:\ManageEngine\EventLog文件夹。对于新的文件夹或默认文件夹不存在，产品将自己创建这些文件夹进行安装。
  • 输入Web服务器端口。默认为8400。确保你要使用的端口没有被其他程序占用。
    选择语言（简体中文、繁体中文、英语、日语或其他）。确保你的浏览器支持选择的语言。
    选择Web协议（HTTP或HTTPS）。
  • 选择安装EventLog Analyzer为服务选项将安装产品为Windows或Linux的服务。该项默认是选择的，你可以不选择该项，以后再安装为服务。我们建议你安装为服务。
  • 输入开始菜单选手的产品目录名称。默认为ManageEngine EventLog Analyzer <版本号>。
  • 输入你个人信息，以便于获得技术支持。

最后，安装向导显示阅读自述文件和启动产品选项。
这样EventLog Analyzer就安装完成了。

说明：EventLog Analyzer提供了三种语言：英语、中文和日文。还有一个选项是“其他”，如果你要EventLog Analyzer支持双字节（UTF-8）语言，请选择此项。

 

如何卸载？

卸载32位和64位版本的方法是相同的。

Windows：

  1. 打开Evenglog Analyzer的开始菜单。默认为开始 > 所有程序 > ManageEngine EventLog Analyzer。

  2. 选择Uninstall EventLog Analyzer。

  3. 在卸载Evenglog Analyzer之前，确认是否卸载。

Linux：

  1. 打开<EventLog Analyzer 主目录>/server/_uninst

  2. 执行命令：./uninstaller.bin

  3. 确认后开始卸载。
