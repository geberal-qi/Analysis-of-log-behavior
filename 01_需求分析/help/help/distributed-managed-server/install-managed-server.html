安装被管服务器

如果您要在Windows系统中安装EventLog Analyzer，执行ManageEngine_EventLogAnalyzer.exe文件，如果在Linux系统中进行安装，则执行ManageEngine_EventLogAnalyzer.bin文件。

如果您要在Windows系统中安装64位的EventLog Analyzer，执行ManageEngine_EventLogAnalyzer_64bit.exe文件，如果在Linux系统中进行安装，则执行ManageEngine_EventLogAnalyzer_64bit.bin文件。

这里有两种安装选项：

  • 一键安装
  • 高级安装
一键安装不能用来将产品安装为被管服务器。

选择高级安装选项，可以自定义产品的安装，向导将会指引您完成安装。

注意：

  1. 请确保您要将安装的被管服务器，连接到的分布式版本管理服务器正在运行。
  2. 请确保在安装被管服务器过程中设置的管理服务器明细正确无误，否则，被管服务器安装会无法完成。管理服务器明细会在安装过程结束时进行验证。

高级安装概述

  • 同意许可协议的条款和条件，您可以将其打印出来以备离线查看。
  • 选择要安装的版本，可选择的版本包括独立安装版（Standalone）、分布式版本（Distributed）和免费版（Free）

    独立安装版 - 如果您要对在单独地理位置下的小型或中型业务进行管理，监视的设备和/或应用数量少于600，那么您可以选用这个版本。

    分布式版本 - 如果您要对在分散的地理位置环境下的大型业务进行管理，监视的设备和/或应用数量少于12000，那么您需要选用分布式版本。

    免费版 - 如果您要对微型业务或SOHO（Small Office and Home Office， 即小型办公室和家庭办公室）进行管理，监视的主机数不超过5台，您可以下载ManageEngine_EventLogAnalyzer独立安装版的exe或bin文件，将其安装为免费版。

  • 选择分布式版本和被管服务器，在被管服务器配置窗口，输入管理服务器主机，保留或修改管理服务器端口，如果管理服务器使用安全通信（HTTPS），则可选择使用HTTPS，或者您也可以不选这个选项。
  • 如果被管服务器需要使用代理服务器，请在代理服务器配置窗口，配置代理服务器主机、代理服务器端口、代理服务器用户名和用户密码信息。
  • 使用浏览按钮，选择产品的安装目录，默认的安装目录为C:\ManageEngine\EventLog。如果新目录或默认目录不存在，它将会自动进行创建，然后开始进行安装。
  • 输入Web服务器端口，默认端口为8400，您也可以指定一个其它的端口，请确保默认端口或您指定的端口没有被其它应用占用。
    选择语言（简体中文-Simplified Chinese、繁体中文-Traditional Chinese、英文-English、日文-Japanese或其它O-thers），请确保您所使用的浏览器支持您所选择的语言。
    选择Web协议（HTTP/HTTPS），使用非安全通信协议HTTP或安全通信协议HTTPS。
  • 选择将EventLog Analyzer安装为服务选项，可以将产品安装为Windows或Linux服务。默认此选项被选中，您可以取消此选项将产品安装为应用，之后您也可以将其变更为服务，我们推荐您将其安装为服务。
  • 输入产品在程序文件夹中显示的文件夹名称，默认它为ManageEngine EventLog Analyzer x文件夹。
  • 输入您的个人信息以便获取帮助和支持

在安装结束的时候，向导将会显示自述文件，并启动EventLog Analyzer服务器。
至此，EventLog Analyzer产品已安装完成。

说明：在Linux系统上进行安装时，如果您的到一个错误消息，显示缓存文件夹没有足够的空间，请尝试执行这个命令 -is:tempdir <directory_name> 选项，<directory_name> 表示现有目录的绝对路径。 ./<file_name>Bin -is:tempdir <directory_name>

 

说明：EventLog Analyzer支持3中语言，即英文、中文和日文，同时还有第四个选项，其它（Other）。如果用户希望 EventLog Analyzer支持双字节（UTF-8）的语言，那么用户可以在安装时选择Other选项。

 

卸载被管服务器

卸载64位和32位版本的EventLog Analyzer是一样的。

Windows：

  1. 定位到EventLog Analyzer安装到的程序文件夹，开始 > 所有程序 > ManageEngine EventLog Analyzer。
  2. 选择卸载选项 Uninstall EventLog Analyzer.
  3. 确认您的操作，继续进行卸载。

Linux：

  1. 定位到 <EventLog Analyzer Home>/server/_uninst 目录。
  2. 执行命令 ./uninstaller.bin
  3. 确认您的操作，继续进行卸载。
说明：在卸载完成之后，卸载反馈表格页面将会打开，我们诚请您提供您卸载产品的原因，帮助我们提升产品的质量。
