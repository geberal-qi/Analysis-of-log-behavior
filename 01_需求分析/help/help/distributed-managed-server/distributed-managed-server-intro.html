EventLog Analyzer分布式版本 - 被管服务器

对于跨地理位置的企业，管理分支机构主机的日志/Syslog通常会遇到这样或那样的问题，为使此项任务可以简单的执行，EventLog Analyzer提供分布式部署方案，它使用分布的模式。

什么是EventLog Analyzer分布式版本？

EventLog Analyzer分布式版本是EventLog Analyzer的一种分布式安装版本，它包括一个管理服务器和多个被管服务器，被管服务器被安装在不同的地理位置（每个地理位置表示一个局域网环境），与管理服务器相连。它可以使网络管理员在一个中心点上，访问不同远程位置的主机信息。所有的报表、告警和其它主机信息，都可以在一个单独的控制台进行访问。 拥有各种分支机构的大型企业的管理员，可以使用这个版本站在全局的立场对业务进行管理。而对于托管安全服务提供商（MSSP），它更是一个福音，他们可以在一个点上对安装在不同客户位置的被管服务器进行监视。

 

EventLog Analyzer分布式版本

  • 对在不同地理位置的整个企业进行聚合日志管理
  • 可扩展的架构，支持1000台主机
  • 使用单一控制台进行集中管理
  • 使用HTTP安全通信
  • 对MSSP的不同客户，专属分段和查看安全化
